2. Historical contextInvented in the 1830s by Samuel F.B. Morse, with help from Alfred Vail and Joseph Henry, to work with the electric telegraph.The first long-distance transmission happened on May 24, 1844, from Washington, D.C. to Baltimore. Morse sent “What hath God wrought?” from the U.S. Capitol.The original American version used variable-length dashes and spaces. In 1851, the Conference of European Nations created International Morse Code, which standardized patterns using only dots and dashes for global use.It became critical for maritime, military, and amateur radio communication. SOS became the international distress signal because it’s short and distinct.It’s largely obsolete for general communication today, but still used in aviation, maritime, and by amateur radio operators. ebsco.comscience.howstuffworks.comstudy.comstudy.comsimple.wikipedia.org
